# 🎨 Anemolif - Pintura y Decoración

Sitio web profesional para el negocio de pintura y decoración **Anemolif**. Una landing page moderna, responsiva y optimizada para convertir visitantes en clientes.

## ✨ Características

- ✅ **Diseño Responsivo** - Perfecto en móvil, tablet y desktop
- ✅ **Galería de Proyectos** - Muestra tus trabajos de forma elegante
- ✅ **Contacto Directo** - Llamada y WhatsApp integrados
- ✅ **Secciones Modernas** - Hero, Galería, Servicios, Contacto
- ✅ **Animaciones Suaves** - Transiciones y efectos profesionales
- ✅ **Optimizado para SEO** - Estructura HTML5 semántica
- ✅ **Rendimiento Rápido** - Imágenes comprimidas automáticamente

## 🚀 Estructura del Proyecto

```
anemolif/
├── index.html          # Página principal
├── styles.css          # Estilos profesionales
└── README.md          # Este archivo
```
